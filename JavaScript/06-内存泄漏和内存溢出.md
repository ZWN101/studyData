### 内存溢出

* 一种程序运行时出现的错误
* 当程序运行需要的内存超过了剩余内存时，就会抛出内存溢出的错误（内存不足）

### 内存泄漏

* 占用的内存没有及时得到释放
* 内存泄漏积累多了容易导致内存溢出
* 常见的内存泄漏

1. 意外的全局变量

````js
function fn1(){
a=0;
console.log(a);
}
fn1();
````

2. 没有及时清理计时器或者回调函数

3. 闭包

```js
function fn1(){
var a1=10;
    function fn2(){
        console.log(++a);
    }
    return fn2;
}
var fn=fn1();
 //用过一次之后不用了，要及时清理掉，不然a不会被释放，会常驻内存中，占用内存
fn();
```
